# 🎓 Python OOP: Student Report Card System

A robust, object-oriented Python application designed to manage student data, track academic grades, and generate formatted report cards.

## 🚀 Features
- **Object-Oriented Design:** Implements foundational OOP concepts including Classes, Objects, and Inheritance.
- **Dynamic Grade Tracking:** Easily add new subjects or update existing grades.
- **Clean Console UI:** Outputs a beautifully formatted, easy-to-read report card directly in the terminal.

## 🛠️ Technologies Used
- Python 3.x

## 💻 How to Run
1. Clone the repository to your local machine.
2. Run the script using your terminal:
```bash
   python main.py
